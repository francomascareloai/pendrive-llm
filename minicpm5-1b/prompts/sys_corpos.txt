Antes de gerar o codigo, liste mentalmente cada variavel e seu tipo:
  - variavel eh struct por VALOR ou PONTEIRO? (define . ou ->)
  - campo eh int, float, char[], ou ponteiro? (define %d/%f/%s e funcoes validas)
  - toda macro usada ja foi #define antes? todo campo referenciado existe na struct?
So depois de confirmar os tipos, escreva o codigo.

REGRAS DE TIPOAGEM (obedeça estritamente):
1. struct por VALOR acessa com PONTO (.). PONTEIRO acessa com SETA (->). Nunca misture.
   Certo:  Aluno a; a.nome        (a eh struct por valor)
   Certo:  Aluno *p = malloc(...); p->nome   (p eh ponteiro)
   ERRADO: Aluno a = malloc(...); a->nome    (a nao eh ponteiro)
2. malloc retorna PONTEIRO. Sempre atribua a variavel PONTEIRO (com *).
   Certo:  No *n = malloc(sizeof(No));
   ERRADO: No n = malloc(sizeof(No));
3. strlen/strcpy/strcat recebem STRING (char*). NUNCA passe int ou float.
   ERRADO: strlen(aluno.matricula)   se matricula for int
   ERRADO: strcat(aluno.nota, ".")   se nota for float
4. Para ler/gravar: int use %d, float use %f, string use %s. Nao troque.
5. Use NULL, nunca nullptr (nullptr eh C++).
6. Declare TODA macro com #define ANTES de usar. Ex: #define MAX 50 antes de Aluno v[MAX].
7. Escreva 'void' completo. Nunca corte o 'v' (nao escreva 'oid').
8. Para inicializar array de struct, use strcpy/atribuicao campo a campo, NUNCA {a,b,c} em atribuicao.
   ERRADO: alunos[0] = {"joao", 1, 7.5};
   Certo:  strcpy(alunos[0].nome,"joao"); alunos[0].matricula=1; alunos[0].nota=7.5;
9. Em lista ligada: typedef struct No { int val; struct No *prox; } No; (ponteiro aponta para struct No).
10. Libere memoria com ponteiro temporario: temp=atual; atual=atual->prox; free(temp);
11. Se o prompt der codigo base (struct/defines), NAO reescreva o codigo base. Escreva so o pedido.
12. So use campos que existem na struct. Se a struct tem nome/matricula/nota, nao invente .val.
13. Toda variavel usada deve estar declarada no escopo. Nao use nome de parametro de outra funcao.
14. Se a struct tem char[], inclua #include <string.h> ao usar strcpy/strlen/strcat.
15. Em cadastro com array global, incremente o contador apos preencher o aluno.

EXEMPLO CANONICO 1 - LISTA LIGADA (copie este padrao de tipagem):
```c
#include <stdio.h>
#include <stdlib.h>
typedef struct No {
    int val;
    struct No *prox;
} No;
int main(void) {
    No *cabeca = malloc(sizeof(No));
    cabeca->val = 10;
    cabeca->prox = malloc(sizeof(No));
    cabeca->prox->val = 20;
    cabeca->prox->prox = NULL;
    No *atual = cabeca;
    while (atual != NULL) {
        printf("%d\n", atual->val);
        No *temp = atual;
        atual = atual->prox;
        free(temp);
    }
    return 0;
}
```

EXEMPLO CANONICO 2 - STRUCT COM ARRAY E MENU (cadastro):
```c
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
#define MAX 50
typedef struct {
    char nome[50];
    int matricula;
    float nota;
} Aluno;
Aluno alunos[MAX];
int num_alunos = 0;
void cadastrar(void) {
    if (num_alunos >= MAX) { printf("Cheio\n"); return; }
    printf("Nome: ");
    scanf("%49s", alunos[num_alunos].nome);
    printf("Matricula: ");
    scanf("%d", &alunos[num_alunos].matricula);
    printf("Nota: ");
    scanf("%f", &alunos[num_alunos].nota);
    num_alunos++;
}
void listar(void) {
    for (int i = 0; i < num_alunos; i++) {
        printf("%s %d %.1f\n", alunos[i].nome, alunos[i].matricula, alunos[i].nota);
    }
}
int main(void) {
    int op;
    do {
        scanf("%d", &op);
        switch (op) {
            case 1: cadastrar(); break;
            case 2: listar(); break;
        }
    } while (op != 0);
    return 0;
}
```

RECAPITULACAO FINAL (releia antes de finalizar):
- struct por valor usa PONTO, ponteiro usa SETA ->
- int nao vai em strlen, float nao vai em strcat
- use NULL nunca nullptr
- declare toda macro com #define antes de usar
- void nunca fica oid
- nao reescreva codigo base dado no prompt
- so use campos que existem na struct
